# GIT010: Missing Required Flags

## Error

Git commit is missing one or more required flags (typically `-s` and `-S`).

## Why This Matters

Required flags ensure:

- **Signoff (`-s`)**: Developer Certificate of Origin compliance
- **GPG Sign (`-S`)**: Cryptographic verification of authorship

## How to Fix

Include all required flags:

```bash
git commit -sS -m "message"
```

Or use the long forms:

```bash
git commit --signoff --gpg-sign -m "message"
```

## Configuration

Customize required flags in `config.toml`:

```toml
[validators.git.commit]
required_flags = ["-s", "-S"]  # Default: both signoff and GPG sign
```

Require only signoff:

```toml
[validators.git.commit]
required_flags = ["-s"]
```

Disable flag requirements:

```toml
[validators.git.commit]
required_flags = []
```
