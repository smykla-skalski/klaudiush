# GIT007: Remote Does Not Exist

## Error

The specified git remote does not exist in this repository.

## Why This Matters

Pushing to a non-existent remote will fail. This validates remotes before attempting the push to provide a clearer error message.

## How to Fix

1. Check available remotes:

   ```bash
   git remote -v
   ```

2. Use an existing remote:

   ```bash
   git push origin branch-name
   ```

3. Or add the missing remote:

   ```bash
   git remote add myremote https://github.com/user/repo.git
   git push myremote branch-name
   ```

## Common Remote Names

| Name       | Typical Usage                       |
|:-----------|:------------------------------------|
| `origin`   | Your fork or primary remote         |
| `upstream` | The original repository (for forks) |
