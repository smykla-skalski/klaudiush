# SEC002: Hardcoded Password Detected

## Error

Detected what appears to be a hardcoded password in the code.

## Why This Matters

- Passwords in code are visible to anyone with repository access
- Git history preserves passwords even after removal
- Shared passwords violate security best practices
- Credential reuse puts multiple systems at risk

## Detected Patterns

This error is triggered by patterns like:

```text
password = "secret123"
passwd: 'mypassword'
pwd='admin123'
```

## How to Fix

1. **Never hardcode passwords**

2. **Use environment variables:**

   ```go
   dbPassword := os.Getenv("DB_PASSWORD")
   ```

3. **Use configuration files (excluded from git):**

   ```yaml
   # .env (in .gitignore)
   DB_PASSWORD=secure_password_here
   ```

4. **Use secret injection at runtime:**
   - Kubernetes Secrets
   - Docker secrets
   - CI/CD secret variables

## Configuration

Allow test passwords:

```toml
[validators.secrets]
allow_list = [
    "password.*test",
    "password.*example",
]
```

Disable the password pattern entirely:

```toml
[validators.secrets]
disabled_patterns = ["generic-password"]
```
