# GIT015: Signoff Identity Mismatch

## Error

The `Signed-off-by` trailer doesn't match the expected identity.

## Why This Matters

- Ensures commits are signed off by the correct identity
- Prevents commits with incorrect author attribution
- Enforces organizational identity policies

## How to Fix

1. Check your git configuration:

   ```bash
   git config user.name
   git config user.email
   ```

2. Update if needed:

   ```bash
   git config user.name "Your Name"
   git config user.email "your.email@example.com"
   ```

3. Ensure the signoff matches:

   ```bash
   # The -s flag uses your configured identity
   git commit -sS -m "message"
   ```

## Configuration

Specify expected signoff in `config.toml`:

```toml
[validators.git.commit.message]
expected_signoff = "Your Name <your.email@example.com>"
```

Or disable signoff identity checking:

```toml
[validators.git.commit.message]
expected_signoff = ""
```
