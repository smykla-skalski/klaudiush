# GIT012: Claude Attribution in Commit Message

## Error

Commit message contains AI-generated attribution (e.g., "Generated by Claude" or "Co-authored-by: Claude").

## Why This Matters

- Commits should be attributed to human developers who review and approve changes
- AI attribution in git history can create legal/compliance complications
- The commit author takes responsibility for the changes

## How to Fix

Remove AI attribution from commit messages:

```bash
# Wrong:
git commit -sS -m "feat(api): add endpoint

Generated by Claude"

# Correct:
git commit -sS -m "feat(api): add endpoint"
```

## Configuration

Disable this check in `config.toml`:

```toml
[validators.git.commit.message]
block_ai_attribution = false
```
